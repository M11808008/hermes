n8n 簡介

n8n（唸作 "n-eight-n"）是一套開源工作流自動化平台，類似 Zapier 但支援自行架設，無需支付訂閱費用。支援：

  • **視覺化流程編輯**：拖放式節點設計
  • **豐富整合**：300+ 預建節點，涵蓋 GitHub、Slack、Notion、資料庫等
  • **自架部署**：Docker、K8s 均可快速部署
  • **程式碼擴展**：可嵌入 JavaScript / Python 自訂邏輯
  • **觸發方式**：Webhook、Schedule、Event、CRON 等

---

使用方式

  • **Clone 專案**：`git clone https://github.com/Zie619/n8n-workflows.git`
  • **匯入 n8n**：在 n8n 介面中 Import 對應的 JSON workflow 檔案
  • **配置憑證**：填入 API Key、連線設定
  • **啟動流程**：activate workflow 即可運行
