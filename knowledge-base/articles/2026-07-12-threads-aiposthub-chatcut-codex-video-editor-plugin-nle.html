AI郵報 Threads｜ChatCut Codex 外掛：把 NLE 影片剪輯搬進 Codex App

一句話摘要

ChatCut 的 Codex / ChatGPT 外掛把「AI 影片剪輯」從聊天建議推向同介面協作：人類在時間軸上確認意圖，Agent 直接操作剪輯專案。

Threads 貼文重點

AI郵報（@aiposthub）指出：

  • Codex 現在連影片都能剪。
  • ChatCut 推出新的 Codex 外掛，把完整的非線性剪輯器 NLE 搬進 Codex App。
  • 安裝後，使用者可以一邊在時間軸上調整素材，一邊讓 Codex 處理剪輯工作。
  • 可執行的工作包含：剪掉不要的片段、重新排列畫面、加入字幕與轉場、調整音訊與節奏、根據文字指令反覆修改。
  • 作者給的範例指令包括：

- 「把前 10 秒剪得更緊湊」

- 「刪掉所有停頓超過 2 秒的片段」

- 「幫我做成 60 秒直式短影音」

  • 貼文提到已付費使用 Codex 的用戶可免費使用這款外掛；安裝提示為：`Turn Codex into a Video Editor, read chatcut.io/codex`。

官方來源交叉驗證

ChatCut 官方網站

ChatCut 首頁標題為「ChatCut · AI Video Editor | Edit Videos with Prompts」，定位為 AI video editor。首頁文案描述：

  • 使用者告訴它想要什麼，它會判斷怎麼做並執行。
  • 可在瀏覽器中使用，也可透過 agent tools 使用。
  • 首頁提供「Use ChatCut in ChatGPT」提示：`Read chatcut.io/chatgpt to install the ChatCut plugin and set up a new task for me.`
  • 頁面呈現時間軸、video/audio tracks、graphics tracks 等 NLE 常見概念。
  • 官方功能文案涵蓋 transcript editing、auto captions、AI image generation、music、motion graphics、exports 等。

ChatCut Codex / ChatGPT 安裝指南

`chatcut.io/codex` 回傳的安裝指南顯示：

  • 安裝的是 ChatCut Codex plugin，公開 marketplace repository 為 `https://github.com/ChatCut-Inc/agent-plugin.git`。
  • 指南特別要求先確認使用環境：若在 web app 或 isolated remote workspace，不能直接修改桌面 App 的 local plugin directory；需在安裝目標機器上的 ChatGPT / Codex desktop app 內操作。
  • 指南要求使用 Codex desktop app bundled Codex CLI，而非隨機 PATH 中的 standalone Codex CLI。
  • 安裝流程包含 marketplace add、plugin list、plugin add、`mcp login chatcut` 與後續驗證。
  • ChatCut MCP server 透過 hosted endpoint 連接：`https://api.chatcut.io/api/external-mcp/mcp`。

GitHub repo

  • Repo 內容包含 `chatcut/`、`.codex-plugin/plugin.json`、`.mcp.json`、skills 與 assets。

更精準的理解

這不是單純「AI 幫你想剪輯建議」，而是 agent 連到剪輯專案與工具介面後，直接參與時間軸操作。關鍵差異如下：

  • 傳統剪輯軟體：人類手動切片、拖移、對齊字幕、調整節奏。
  • 一般 AI 聊天：AI 給剪輯建議或腳本，但仍由人類回到剪輯軟體執行。
  • ChatCut + Codex 外掛：人類用自然語言描述目標，Agent 可在 ChatCut 專案中執行剪輯、字幕、轉場、輸出與可視化驗證。

這也讓 Codex 的邊界從 coding agent 擴張到 creative production agent：它不只改 repo，也能透過 plugin / MCP 操作媒體創作工具。

實務落地建議

適合嘗試的場景：

  • 長訪談或課程影片初剪：刪除停頓、口頭禪、空白片段。
  • 短影音轉製：把橫式長片整理成 60 秒直式版本。
  • 教學影片：自動產生字幕、章節、重點畫面與 B-roll 提示。
  • 社群內容 repurpose：同一素材輸出多版本比例與長度。
  • 團隊剪輯 SOP：把常見規則寫成可反覆執行的 prompt / skill。

使用時應保留人工審核點：

  • 確認 cuts 是否破壞語意。
  • 檢查字幕錯字、專有名詞與時間軸同步。
  • 檢查音訊 loudness、轉場節奏與版權素材。
  • 輸出前確認平台比例、解析度、bitrate、檔名與封面。

對 AI 學習雷達的價值

這則貼文值得高優先保存，因為它代表 agent plugin 的應用場景正在越過程式碼：

  • Coding agent 變成「能操作專業工具的工作流 agent」。
  • MCP / plugin marketplace 讓外部 SaaS 可被 agent 安全接入。
  • 影片剪輯這種高度時間軸、視覺與感覺導向的工作，也開始被自然語言與可視化驗證重組。
  • 未來課程與內容生產流程，可以把「拍攝 → 初剪 → 字幕 → 轉製 → 匯出」拆成 agent 可協作的子任務。

注意事項

  • 貼文中的「已付費使用 Codex 的用戶可免費使用外掛」需以 ChatCut 與 Codex 當下方案政策為準；ChatCut 本身仍可能有帳號、credits、匯出與功能限制。
  • 安裝指南明確區分 web app / isolated workspace 與 desktop app；若環境不對，agent 無法替使用者安裝本機外掛。
  • 官方指南要求使用 desktop app bundled Codex CLI，不應直接拿 standalone npm / Homebrew Codex CLI 代替。
  • 影片剪輯輸出仍需人工 QC，尤其是語意切割、字幕、版權、音量與平台規格。
