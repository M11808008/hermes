Threads｜Marble 開放小學課程 DAG：1,590 個微主題與先備知識圖譜，可作教育 Agent 學習地圖

一句話摘要

Marble 的 os-taxonomy 將小學階段知識拆成微主題節點與先備知識邊，讓教育 agent 可以從「內容推薦」升級為「依先備能力規劃學習路徑」。

Threads 貼文重點

@heroman1916 提醒台灣 AI 工程師與創作者：Marble 把小學課程，例如數學、科學、運算與生活技能，整理成開放 DAG 圖譜，並放在 GitHub 可直接下載。貼文指出，這可作為教育 agent 或兒童 AI 工具的可靠學習地圖。

作者在延伸留言補充，這套資料的目標群組是小學／初等階段「學什麼」的微主題知識圖譜，例子包括：

  • 辨識日常生活中的 AI，例如手機人臉解鎖、影片推薦、自動門。
  • 能正確計算 2 位數加法並解釋過程。
  • 植物、動物生命週期。

資料結構與可用檔案

README 列出的核心資料在 `data/`：

  • `data/topics.json`：micro-topics，也就是圖譜節點。
  • `data/dependencies.json`：prerequisite edges，也就是某主題依賴哪些先備主題。
  • `data/curriculum-standards.json`：來源課綱標準。
  • `data/manifest.json`：檔案大小、SHA-256、計數與生成資訊。
  • `data/clusters.json`：主題群組摘要。

每個 micro-topic 不只是標題，還包含 plain-language description、mastery evidence criteria、type、subject、domain、age range，以及對應課綱來源。這讓它比一般課綱條列更適合被 agent 查詢與推理。

對教育 Agent 的價值

這份資料可作為教育 agent 的四層基礎：

  • 診斷層：學生卡在某題時，回推可能缺少哪些先備概念。
  • 路徑層：從目標能力往前展開 prerequisite chain，產生學習順序。
  • 內容層：為每個 micro-topic 生成練習、講解、實驗或互動範例。
  • 追蹤層：把 mastery evidence 轉成檢核點，讓 agent 不只聊天，而能追蹤能力是否真的掌握。

對兒童 AI 工具尤其重要，因為兒童學習不是單純「問什麼答什麼」；它需要知道學生現在在哪、下一步補什麼、哪些概念不能跳過。

可落地的應用想法

  • AI 家教：輸入學生錯題，回推先備知識並生成補強練習。
  • 課程設計：用 DAG 自動生成 4 週、8 週或一學期學習路徑。
  • 互動教材：把每個 micro-topic 接到一個模擬、故事或遊戲任務。
  • 親子學習儀表板：用節點圖顯示孩子已掌握、待補強、下一個可挑戰主題。
  • 台灣課綱對齊：以 Marble DAG 為骨架，再映射台灣國小領綱與年級能力指標。

使用限制與注意事項

  • Marble 的資料主要根據美國與英國課綱，不能直接等同台灣國小課綱。
  • DAG 是學習路徑骨架，不是完整教材；仍需教案、例題、評量與在地語境。
  • 先備關係可能有 hard / soft 之分，教育 agent 實作時不應把所有邊都當作絕對門檻。
  • 若做兒童產品，需特別處理個資、學習紀錄、家長同意與內容安全。
  • 授權包含開放資料義務，商用或再發布前應確認 ODbL / CC-BY-SA 的 share-alike 與 attribution 要求。
