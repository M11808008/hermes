open-slide｜為 AI Agent 設計的開源簡報框架，可用自然語言生成 React 簡報

來源：Threads@spedmix2025；專案：1weiho/open-slide

一句話摘要

open-slide 是一個為 AI coding agent 設計的開源簡報框架，讓使用者用自然語言描述主題與風格，再由 Agent 生成 React 簡報；它負責固定畫布、縮放、導覽、present mode、熱更新與匯出，特別適合教學、課程與 AI 輔助簡報製作。

Threads 貼文重點

作者分享自己使用 open-slide 製作「國一統計圖表」簡報的經驗，認為它是目前用過最方便的簡報方式，亮點包括：

  • 可以修改
  • 內建精美的粉筆動態效果
  • 可以匯出 PDF
  • 可透過 Vercel 發布成永久免費的網頁版簡報連結
  • 不需要程式基礎，也可把 GitHub 專案網址交給 Antigravity，請它協助製作與部署
  • 作者表示 Codex 理論上也可以跑，但在他的電腦上 Antigravity 較快

GitHub 專案資訊

open-slide 官方描述是：A slide framework built for agents.

它的核心概念是：Slides are visual code. Agents are great at writing code.

也就是把簡報視為視覺化程式碼，讓 Claude Code、Codex、Cursor、Antigravity 等 coding agent 負責寫 React，而 open-slide 提供穩定的簡報 runtime。

截至擷取時，GitHub 約有：

  • 4.6k stars
  • 316 forks
  • MIT License
  • 技術主體為 TypeScript / React

核心能力整理

| 能力 | 說明 |

|---|---|

| Agent-native authoring | 可用自然語言要求 Agent 建立整份簡報 |

| React slide pages | 每頁都是任意 React component，而不是受限 DSL |

| 固定畫布 | 每頁渲染為 1920 × 1080 canvas，適合穩定輸出簡報比例 |

| Built-in skills | 內建 /create-slide、/slide-authoring 等技能，協助代理規劃與寫作 |

| In-browser inspector | 可點選元素加註修改意見，再讓 Agent 套用 comments |

| Assets manager | 管理圖片、影片、字體，並整合 svgl logo 搜尋 |

| Present mode | 支援全螢幕播放、鍵盤導覽、presenter mode、speaker notes 與 timer |

| 匯出 | 可輸出自包含 static HTML 或 print-ready PDF |

| 部署友善 | 可部署到 Vercel、Cloudflare Pages、Netlify、Zeabur 等靜態站 |

使用方式

官方 quickstart：

```bash

npx @open-slide/cli init my-slide

cd my-slide

pnpm dev

```

貼文作者的實務流程則更適合非工程背景使用者：

  • 把 open-slide GitHub 網址丟給 Antigravity。
  • 直接描述想做的簡報主題，例如「國一統計圖表」。
  • 讓 Agent 生成簡報與程式碼。
  • 需要時再要求 Agent 修改內容或樣式。
  • 發布到 Vercel，取得可分享的網頁版簡報連結。
  • 需要交付或列印時匯出 PDF。

為什麼值得收藏

1. AI 簡報不再只是 PowerPoint 外掛

open-slide 的路線不是把 AI 包進 PowerPoint，而是把簡報變成可由 Agent 編寫、版本管理、部署與匯出的 web artifact。這讓簡報具備：

  • 程式碼可控性
  • GitHub 版本管理
  • 網頁部署能力
  • 可互動與可動畫的設計空間
  • 可由 Agent 反覆修改的工作流

2. 對教育場景很實用

作者用「國一統計圖表」做示範，這點很值得收錄。教育簡報常見需求是：

  • 要美觀但可快速改
  • 要能輸出 PDF
  • 要能分享網頁連結
  • 要能把抽象概念視覺化
  • 教師不一定具備前端背景

open-slide + AI Agent 的組合，剛好可把「教師描述教學目標」轉成「可呈現的互動式簡報」。

3. 和 Luarn-Pin 的知識庫、課程網頁工作流相容

這類工具可延伸到：

  • AI 課程簡報
  • 教學活動投影片
  • 報告型 web deck
  • 可發布的課程 landing / slide hybrid
  • 用 GitHub Pages 或 Vercel 存放的可重複使用簡報模板

注意事項

  • open-slide 是程式碼型簡報框架，仍需 Node.js / pnpm / Vercel / GitHub 等基本環境。
  • 非工程使用者可透過 Antigravity、Claude Code、Codex 或 Cursor 代勞，但仍要留意部署權限與 GitHub 帳號設定。
  • 因每頁是 React component，彈性很高，但也代表維護方式更接近前端專案，而不是傳統 PPT 檔案。
  • 若需要交給完全不熟網頁環境的人共同編輯，仍需評估協作門檻。
