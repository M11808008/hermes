Runway MCP｜在 ChatGPT 內直接生成與編輯影片、圖片的創作工作流

來源為 Runway 官方 X 貼文。貼文主軸是：可以直接在 ChatGPT 裡使用 Runway 生成與編輯影片、圖片，不必在 ChatGPT 與 Runway 網站之間來回切換。本文以 X 貼文作為發現來源，並以 Runway 官方 MCP 頁面與產品頁補強實際工作流、設定重點與限制。

一句話摘要

Runway MCP 把 Runway 的影片／圖片生成能力接進 ChatGPT、Claude、Cursor、Replit 等支援 MCP 的代理工具中，讓使用者可在原本的對話或開發環境裡請 AI 產生、修改、延伸視覺素材；這代表創作流程正從「切換工具」轉向「代理在同一上下文中調用專業生成工具」。

這件事為什麼重要

傳統 AI 影片工作流通常是：在 ChatGPT 寫 prompt → 複製到 Runway → 生成影片 → 回到 ChatGPT 修 prompt → 再貼回 Runway。Runway MCP 的意義在於把這段切換壓縮成同一個對話上下文：ChatGPT 或其他代理可以直接呼叫 Runway 生成素材，使用者保留「用自然語言描述需求」的體驗，同時使用 Runway 的專業影片與圖片生成能力。

這對創作者、課程設計、廣告企劃與 AI workflow 教學都很有價值，因為它把「文字企劃 → 視覺生成 → 修改迭代」變成一條更連續的 pipeline。

可用模型與能力範圍

Runway 官方 MCP 頁面提到，代理可用的模型會依 Runway 帳號方案而定，包含 Seedance 2.0、Kling、Gen-4.5、Veo、GPT image 2、Nano Banana Pro 等；代理可以自行選模型，也可以由使用者指定模型。

| 能力 | 可能用法 |

|---|---|

| Text / image / video / audio to video | 從文字、參考圖、既有影片或音訊生成影片素材 |

| Image generation | 在 ChatGPT 對話中直接產生分鏡圖、概念圖、廣告素材 |

| Video editing | 請代理依語意修改片段、調整畫面風格或產生變體 |

| Creative workflow context | 把企劃、腳本、分鏡、品牌語氣與生成請求保留在同一段對話中 |

| Team production | 官方頁面表示 MCP 使用與 Runway app 相同系統，作品保存於 library，適合團隊後續管理 |

對欒帥的實用價值

  • 可作為 AI 課程中的「MCP × 多媒體生成」案例：ChatGPT 不只回答文字，也可呼叫專業影片工具。
  • 適合示範 agent workflow：企劃、腳本、分鏡、生成與修圖修片都在同一上下文中迭代。
  • 可和 AI 學習雷達／Hermes 流程結合：Hermes 整理靈感與腳本，ChatGPT/Claude 透過 Runway MCP 生成素材，最後再回到知識庫歸檔成果與 prompt。
  • 對行銷內容與課程預告尤其有用：可快速把文字主題轉成短片、封面圖、分鏡草稿與 social creative。
  • 這類工具適合作為「能力展示」，但正式商用仍要建立人工審核、品牌一致性、授權與成本控管流程。

建議測試流程

| 步驟 | 做法 |

|---|---|

| 1. 設定 MCP | 在 ChatGPT / Claude / Cursor 等支援 MCP 的工具中加入 https://mcp.runwayml.com/mcp，並登入 Runway 帳號 |

| 2. 準備任務 | 用一段明確企劃：目標觀眾、影片長度、風格、主題、品牌限制 |

| 3. 指定模型或讓代理選擇 | 若要可控性，先指定模型；若是快速探索，可讓代理依需求選擇 |

| 4. 生成第一版 | 先生成短片段或圖片，不要一開始就做長片 |

| 5. 對話式迭代 | 要求「縮短前 2 秒」「改成更科技感」「保留構圖但換色調」等具體修改 |

| 6. 記錄成本與品質 | 記錄消耗 credits、等待時間、可用率、是否需要人工後製 |

風險與注意

  • MCP 只是把工具接進代理環境，不代表所有影片生成品質都自動穩定；仍需看模型、素材、提示與方案。
  • 生成會消耗 Runway credits，成本與模型、解析度、長度及設定有關。
  • 若素材涉及人物肖像、品牌 Logo、音樂、新聞片段或商業素材，仍需人工做授權與合規檢查。
  • Agent 自動選模型很方便，但正式專案建議指定模型與輸出規格，避免成本與風格不可控。
  • ChatGPT / Claude / Cursor 等外部代理的 MCP 支援程度、權限設定與檔案保存方式可能不同，需逐一測試。

判讀限制

本文根據 Runway 官方 X 貼文、Runway MCP 官方頁面與產品頁整理；X 貼文中的影片示範未逐幀拆解，Runway help center 的連線說明頁面受 Cloudflare 驗證阻擋，本文未引用其細節。實際支援模型、ChatGPT MCP 入口與方案費用可能隨 Runway 與各代理平台更新而變動，部署前需以官方頁面與帳號內設定為準。
