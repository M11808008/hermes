Codex Windows 版支援連結手機 ChatGPT + Computer Use 功能更新

重點摘要

作者更新 Windows 版 Codex App 時發現兩項重大更新：

### 1. Windows 版 Codex 支援連結手機 ChatGPT

> Windows 版的 Codex 終於也能連線到手機的 ChatGPT 了！

這項更新讓作者可以跨三台電腦使用 AI Agent：

  • 家用桌機（Windows）
  • 外出講課用筆電（Windows）
  • MacBook（過去主要的本地模型載體）

過去因 Windows 版 Codex 無法直接使用手機 ChatGPT，作者需要把檔案拷貝到 MacBook 測試。現在這個問題已解決。

### 2. Computer Use 功能同步更新

同步推出的 Computer Use 功能，讓 Codex 能直接控制瀏覽器（類似 Anthropic 的 Computer Use），作者表示：

> 雖說之前更新的 Chrome 外掛程式對我來說就已經很夠用了，但連這個都有的話，Codex 能玩的東西就更多了。

應用場景

| 情境 | 之前 | 現在 |

評論

Codex 的 Windows 版持續追上 macOS 版的功能完整性，加上 Computer Use 功能的加入，Codex 作為 AI Coding Agent 的實用性又往前跨了一步。
