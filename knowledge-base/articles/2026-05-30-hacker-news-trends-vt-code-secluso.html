1️⃣ VT Code — 終端機 AI 大起義

**VT Code** 是一個開源 Rust 專案，把大模型直接塞進終端機。

| 項目 | 說明 |

|------|------|

| **GitHub** | https://github.com/vinhnx/vtcode |

| **語言** | Rust（高效能） |

| **核心能力** | 整合 tree-sitter 語法樹 + AST-grep |

| **本地模型支援** | Ollama、LM Studio |

| **特點** | 100% 離線本地運行、Auto Debug |

**被拿來和 Cursor、Claude Code 做比較**，被稱為「最硬核的開源對手」，原因是：

  • 結合 AST 語法樹分析，代碼理解深度超越純 token-based 推理
  • 完全開源，無需訂閱
  • 離線本地跑隱私無憂

2️⃣ Secluso — 居家安防大革命

**Secluso** 是一個開源居家防衛系統，用樹莓派自製安防。

| 項目 | 說明 |

|------|------|

| **硬體需求** | 樹莓派（Raspberry Pi）|

| **加密方式** | 端到端加密（E2EE） |

| **AI 辨識** | 100% 在樹莓派機身內完成，雲端中轉伺服器**完全看不到畫面** |

| **定位** | 隱私主權回歸用戶，避開雲端監視器後門爭議 |

適合想要自己架設安防系統但又不想把隱私交給雲端服務商的技術用戶。
