GitHub：

https://github.com/aqua5230/usage

用途：

把 Claude Code 跟 Codex 的用量直接顯示在 macOS 右上角選單列，避免一直切視窗查額度。

核心功能：

  • Claude Code 用量追蹤
  • 顯示目前 session 用量
  • 顯示 weekly 用量
  • 透過 Claude Code 的 statusLine hook 讀取本機狀態檔
  • 不呼叫 Anthropic API
  • Codex 用量追蹤
  • 讀取 ~/.codex/sessions/ 裡的 jsonl 對話紀錄
  • 從紀錄中的 rate_limits 與 token 統計抓資料
  • 可估算今日 token 與成本
  • 不呼叫 OpenAI API
  • 專案用量卡
  • 可以看各專案用量
  • 可切換今日 / 7 日 / 月
  • 隱私設計
  • 只讀本機檔案
  • 不讀 Keychain
  • 不會偷用 Claude / Codex 額度
  • 唯一可能連網的是抓 LiteLLM 公開價格表，用於 Codex 成本估算；也有 fallback 價格
  • v0.5.0 / 目前 repo 顯示 v0.6.0 方向
  • 面板改成 HTML/CSS 架構
  • 為多面板與未來 Windows 版鋪路
  • 加入多語系：繁中、簡中、英文、日文、韓文
  • GitHub 上目前 stars 約 84、fork 約 14

安裝方式大致有兩種：

一般使用者：

到 GitHub Releases 下載 usage.app.zip，解壓後拖到 Applications。

開發者：

git clone https://github.com/aqua5230/usage.git

cd usage

python3 -m venv .venv

source .venv/bin/activate

pip install -e .

python3 main.py --setup

python3 main.py
