Google AI Pro / Ultra｜每月可領 Gemini API 折抵金，Pro 10 美元、Ultra 最高 100 美元

來源：電腦王阿達 Rocky

一句話摘要

Google AI Pro 與 Google AI Ultra 訂閱用戶可以到 Google Developer Program 領取每月 Google Cloud 折抵金，這筆額度可用於 Google AI Studio / Gemini API 等開發測試，對常用 Gemini 模型跑 Hermes Agent、Agent workflow 或 AI prototype 的人很實用。

重點整理

1. 這是什麼福利？

電腦王阿達文章指出，Google 已把部分 Google Developer Program 進階開發者福利整合到 Google AI Pro / Google AI Ultra 訂閱中。符合資格的使用者可以領取每月 Google Cloud 折抵金，用於測試或開發 AI 應用。

Google 官方 Google Developer Program 頁面也提到，Google AI Pro 或 Ultra 用戶可取得 monthly Cloud credits、較高的 Antigravity quota、Gemini 模型強化存取等開發者福利。

2. 每月額度

| 訂閱方案 | 每月折抵金 | 備註 |

|---|---:|---|

| Google AI Pro | 10 美元 | 可用於 Google Cloud 帳單帳號，文章實測可折抵 Gemini API 花費 |

| Google AI Ultra 20TB | 40 美元 | 文章提醒 Ultra 額度依方案不同，不一定都是 100 美元 |

| Google AI Ultra 30TB | 100 美元 | 最高每月 100 美元折抵金 |

重點是：如果本來就已經訂閱 Google AI Pro / Ultra，這筆 API 額度很容易被忽略，但對開發者來說相當於每月固定的 Gemini API 測試預算。

3. 可以用在哪裡？

文章實測重點是 Gemini API。實務上可對應到：

  • Google AI Studio / Gemini API 呼叫
  • Hermes Agent 中以 Gemini 模型執行長任務或副模型任務
  • Agent automation、資料整理、文件摘要、原型測試
  • 用 Google Cloud billing account 計費的 AI 開發測試

對 Luarn-Pin 的使用情境來說，最有價值的是把這筆額度視為「每月可消耗的 Gemini API sandbox」，用來測試 Hermes Agent、多代理工作流、內容擷取與知識庫建置，而不用一開始就完全吃自費 API 預算。

領取流程

文章提供的流程如下：

  • 進入 Google Developer Program 網站：`https://me.developers.google.com`
  • 點選「Join the program」加入計畫。
  • 建立 Developer Profile；基本上填姓名即可，其餘偏好可視情況取消勾選。
  • 進入左側「福利」。
  • 找到「Unlock Google AI Pro or Ultra」相關福利。
  • 選擇要套用折抵的 Google Cloud billing account。
  • 點選 Apply。
  • 到 Google Cloud 後台確認折抵金已出現。

文章也提醒，如果頁面出現「Join Google One」，可先點開 Google One 方案頁後關閉，再回到福利頁面，該提示可能會消失並顯示可套用的折抵福利。

為什麼值得收藏

1. Gemini API 成本可以被更精準控管

對 AI 工具重度使用者來說，Gemini API 很常被放進 Agent、摘要、批次處理、程式輔助或資料萃取流程。每月 10 / 40 / 100 美元折抵雖不一定足以支撐大規模 production，但很適合用作：

  • prototype 測試額度
  • 個人知識庫與內容 pipeline 的月度消耗
  • Hermes Agent 任務的備用模型預算
  • AI 課程、demo、教學素材的開發測試

2. Google 正把「訂閱制 AI」與「開發者雲端額度」綁在一起

這個福利反映一個趨勢：AI 訂閱不再只是聊天介面，而是逐漸包含模型 API、開發者工具、雲端額度與 Agent 開發平台。未來 Google AI Pro / Ultra、OpenAI ChatGPT 訂閱、Anthropic Claude 訂閱與各家雲端開發者方案，可能都會往「個人 AI 工作台 + API / Agent 開發資源」整合。

注意事項

  • 折抵金需要連到 Google Cloud billing account，仍應確認是否有其他未被折抵的付費資源。
  • Ultra 額度依方案不同：20TB 為每月 40 美元，30TB 才是每月 100 美元。
  • 福利、資格與金額可能會依地區、帳號類型或 Google 政策調整，正式使用前應以 Google Developer Program 福利頁實際顯示為準。
  • 建議在 Google Cloud 設定預算警示，避免測試 Gemini API 時超出折抵金。
