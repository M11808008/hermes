Smart4A Hermes Agent 完整指南：安裝、記憶、技能與 Session 隔離

一句話摘要

Smart4A 的 Hermes Agent 中文完整指南整理了 Hermes Agent 的定位、安裝平台、2026 版本重點、自進化機制、記憶系統、Skill 文件與跨平台 Session 隔離，適合作為中文教學與導入 Hermes Agent 的快速參考。

重點整理

  • 網站標題：Hermes Agent · 安裝速查。
  • 主軸：如何正確設定 Hermes Agent。
  • 定位：Nous Research 開源、會自我進化的 AI Agent，支援 macOS、Linux / WSL2、Windows beta、Web UI、本機 LLM，MIT 開源。
  • 文章比較 Hermes Agent 與 OpenClaw：

- OpenClaw 偏執行與編排、廣度整合、社群技能多、TypeScript。

- Hermes Agent 偏記憶與成長、深度進化、40+ 內建技能與 Agent 自動生成技能、Python。

  • Hermes 的三個核心：

- 跨會話持久化記憶：FTS5 全文索引。

- 任務沉澱技能：複雜任務後自動建立 Skill Markdown。

- 使用者建模：Honcho 辯證式建模，逐步理解工作風格與偏好。

  • v0.13.0 Tenacity Release 重點包含：

- Multi-agent Kanban。

- /goal 長期任務目標。

- Checkpoints v2。

- Gateway auto-resume。

- Cron no-agent watchdog。

- Google Chat。

- Video analyze 工具。

- xAI Custom Voices。

- 多語介面。

  • 安全重點：PII redaction 預設開啟、Discord role allowlist guild-scoped、WhatsApp 預設拒絕陌生人、TOCTOU 漏洞修補。
  • 自進化不是改模型權重，而是把成功路徑蒸餾成技能文件、寫入全文索引，並透過 LLM 摘要偏好。
  • 學習閉環：Observe → Extract → Skill Creation → Refinement → Nudge。
  • 三層記憶：

- 跨 Session 記憶：FTS5 搜尋與摘要索引。

- Skill Documents：把複雜任務提煉成可重用 Markdown 技能。

- User Modeling：透過 Honcho 建模使用者偏好。

  • Session 隔離重點：

- Agent 本體、API Key、設定、模型、MCP 工具、Skill、同機檔案系統可共用。

- 對話歷史與即時記憶各 session 獨立。

- FTS5 與 Honcho 屬於「長期半共用」：會被索引/累積，但不是每輪自動載入所有歷史。

對 Luarn-Pin 的意義

這份指南與 Luarn-Pin 目前使用 Hermes Agent、LINE gateway、AI 學習雷達、本機知識庫管理介面、skills 與跨平台記憶的實作高度相關。它可以作為中文教學材料，也能幫助向學員或合作夥伴解釋 Hermes 與 OpenClaw 的差異、Hermes 為什麼適合做長期個人 AI 作業系統，以及為什麼 Session 隔離不是缺陷，而是合規與隱私上的設計選擇。
