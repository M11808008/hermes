台北信義吳興街口袋小吃｜蛋餅、辣麵、米糕、豆花、雞蛋糕五店路線

來源：欒帥提供的信義區吳興街口袋小吃清單。本文整理成可直接走訪的「北醫 / 吳興街商圈」小吃路線，並用 Brave Search 補查電話、招牌品項與外部食記佐證。

一句話摘要

吳興街是信義區很適合「不想只吃百貨美食街」時使用的在地小吃路線：早餐吃白暮蛋餅，午晚餐選潮饌麵府或陳記米糕，下午用藤香厚工豆花、281雞蛋糕收尾；適合台北101/世貿、北醫、信義區辦事後順路收藏。

店家筆記

1. 白暮蛋餅先生5號店｜北醫一帶早午餐

白暮蛋餅先生5號店位在吳興街284巷，靠近北醫生活圈。外部食記多提到招牌是脆皮蛋餅，包含皮蛋蛋餅、棉花糖起司蛋餅、明太子、薯餅塔等創意早餐品項。

| 重點 | 內容 |

|---|---|

| 類型 | 早餐、早午餐、脆皮蛋餅 |

| 推薦方向 | 皮蛋蛋餅、棉花糖起司蛋餅、薯餅塔、脆皮蘿蔔糕 |

| 使用建議 | 適合早上先吃；如果要串整條吳興街，這間是路線起點 |

| 注意 | 營業時間不同食記略有差異，週末時間需再確認 |

2. 潮饌麵府｜辣麵與米苔目

潮饌麵府在吳興街83號，外部食記描述它以川味擔擔麵、酸辣米苔目、紅油抄手在吳興街商圈小有名氣，是上班族與附近住戶會用的麵館。

| 重點 | 內容 |

|---|---|

| 類型 | 麵館、擔擔麵、米苔目、紅油抄手 |

| 推薦方向 | 擔擔湯麵、擔擔乾麵、擔擔乾米苔目、酸辣米苔目、紅油抄手 |

| 使用建議 | 適合正餐；午餐時段建議避開尖峰或提早到 |

| 注意 | 週日公休，且營業分午晚兩段 |

3. 陳記米糕｜米糕與排骨酥湯

陳記米糕是吳興街150號的平價小吃，外部食記常提到筒仔米糕、排骨酥湯，也有人推薦一人份佛跳牆。對想吃傳統台式飯湯類的人很實用。

| 重點 | 內容 |

|---|---|

| 類型 | 米糕、排骨酥湯、佛跳牆、傳統小吃 |

| 推薦方向 | 筒仔米糕、排骨酥湯、一人份佛跳牆 |

| 使用建議 | 適合午晚餐，不想吃麵時選這間 |

| 注意 | 週六公休；使用者提供 21:00 結束，部分食記列 22:30，建議以現場/Maps 為準 |

4. 藤香厚工豆花｜飯後甜點

藤香厚工豆花在吳興街113號，外部食記形容豆花口感厚實、豆香濃，有豆漿豆花、焦糖、連藕等組合。它的位置也很適合接在潮饌麵府或陳記米糕後面當甜點。

| 重點 | 內容 |

|---|---|

| 類型 | 豆花、甜點、下午茶 |

| 推薦方向 | 厚工豆花、豆漿豆花、焦糖口味、連藕豆花 |

| 使用建議 | 午餐後或晚餐後收尾，六日 09:00 開更早 |

| 注意 | 甜點店資訊相對穩定，但仍建議出發前看 Google Maps |

5. 281雞蛋糕｜吳興街下午點心

281雞蛋糕在吳興街281巷口附近，主打包餡雞蛋糕與可直接買整片的雞蛋糕，外部食記提到口味多、有甜有鹹，常有排隊與提早售完情況。

| 重點 | 內容 |

|---|---|

| 類型 | 雞蛋糕、銅板點心、包餡甜點 |

| 推薦方向 | 整片雞蛋糕、包餡口味、甜鹹口味 |

| 使用建議 | 下午 14:30 後作為散步點心；想吃熱門口味不要太晚到 |

| 注意 | 路邊攤型態，售完可能提早休；地址以「吳興街281巷口12號旁」較好找 |

建議路線

路線 A：早餐到下午茶

白暮蛋餅先生5號店 → 北醫 / 吳興街散步 → 藤香厚工豆花 → 281雞蛋糕

適合不想吃太重、想把吳興街當一段輕食散步的人。

路線 B：正餐小吃線

潮饌麵府 → 藤香厚工豆花 → 281雞蛋糕

適合午餐或晚餐，主食、甜點、點心都有。

路線 C：傳統台式飯湯線

陳記米糕 → 藤香厚工豆花 → 281雞蛋糕

適合想吃米糕、排骨酥湯、佛跳牆這類傳統小吃的人。

  • 本筆記以使用者提供清單為主，Brave Search 補充外部食記與電話；營業時間可能因店家臨時休息、售完或平台更新而不同。
