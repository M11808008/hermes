核心功能

### 1. 極度逼真的語音複製

生成一個熟悉的女聲，溫柔地講著話，連**換氣的停頓**和**喉嚨的微小震動**都一模一樣，結果竟然是 AI 算出來的。

這不是那種聽兩秒就出戲的死板朗讀——這是最近在開源圈瘋傳的 **VoxCPM2**。

---

### 2. 突破性技術：拔掉傳統 Tokenizer

以前總覺得 AI 聲音少了點靈魂，但 VoxCPM2 直接**拔掉傳統的 tokenizer**，在**連續空間裡運算**，把人類聲音裡最細微的情緒全部保留下來了。

免費開源，連**廣東話、四川話**都講得像巷口阿姨一樣自然。

---

### 3. 憑空「捏」出聲音（Text-to-Voice）

你不用拿麥克風錄半天，只要在鍵盤上敲幾個字：

> - 「一個 30 歲、聲音溫柔帶點沙啞的女性」

> - 「海盜船長的粗啞嗓音」

滑鼠點下去，幾秒後，帶著顆粒感、甚至微微透著疲憊的真實嗓音就出來了。

2026-06-20 補充：Govin.AI Threads 與官方 Repo 校正

欒帥再次分享 Govin.AI 的 Threads 貼文，該貼文指向官方 GitHub `OpenBMB/VoxCPM`，並強調 VoxCPM2 可用文字描述產生聲線、用短參考音訊做 voice cloning、48kHz 輸出、Apache-2.0 授權與商用可用性。

本次以 GitHub API 與官方 README 校正後，應記錄為：

| 項目 | 官方/目前狀態 |

|---|---|

| Repo | https://github.com/OpenBMB/VoxCPM |

| 官方描述 | VoxCPM2: Tokenizer-Free TTS for Multilingual Speech Generation, Creative Voice Design, and True-to-Life Cloning |

| 授權 | Apache-2.0 |

| 語言/技術 | Python、PyTorch、MiniCPM-4 backbone |

| 模型規格 | VoxCPM2 為 2B 參數模型，官方稱使用超過 200 萬小時 multilingual speech data 訓練 |

| 語言支援 | 官方 README 列 30 種語言，另列中文方言：四川話、粵語、吳語、東北話、河南話、陝西話、山東話、天津話、閩南話 |

| 功能 | Voice Design、Controllable Voice Cloning、Ultimate Cloning、context-aware synthesis、48kHz audio output |

| 即時效能 | README 稱 RTX 4090 上 PyTorch RTF 約 0.3；Nano-vLLM / vLLM-Omni 可到約 0.13 |

| GitHub snapshot | 2026-06-20 約 30.8k stars、3.4k forks、122 open issues（社群數字會快速變動） |

對欒帥的實作判斷

  • 若只是做短影片旁白、Podcast 試配音，先用 Hugging Face demo 或社群 demo 驗聲線與口音，不必一開始自架。
  • 若要穩定量產，重點不是「模型免費」，而是 GPU / CUDA / queue / 音檔儲存 / 濫用防護 / 授權告知與聲音同意流程。
  • Apple Silicon 可嘗試 MPS/CPU，但官方即時效能數字主要以 NVIDIA RTX 4090 與 vLLM/Nano-vLLM serving 為參考；不要把社群貼文的「免費」誤讀成零算力成本。
  • Voice cloning 涉及冒用與詐騙風險，公開產品應加入使用者授權、浮水印或明確 AI 生成標示。
