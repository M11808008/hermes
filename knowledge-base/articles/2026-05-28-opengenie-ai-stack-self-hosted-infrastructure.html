OpenGenie AI Stack — 自託管 AI 基礎設施，幾分鐘即可部署

**來源：** [TigerAI Taiwan · GitHub Pages](https://tigerai-taiwan.github.io/OpenGenie-AI-Stack/) · MIT License

---

專案定位

> Self-Hosted AI Stack Ready in Minutes

一個模組化、生產級的自託管 AI 基礎設施框架，支援 **AMD ROCm、NVIDIA CUDA、ARM64/Apple Silicon** 三大硬體平台，透過 Docker Compose 在**十幾分鐘**內完成從零到完整 AI 系統的部署。

**⚠️ 注意：目前支援 Ubuntu 22.04 / 24.04**，尚未支援 macOS（即使 Apple Silicon 也需透過 Docker Desktop Linux VM）。

---

12 階段部署架構

每個 Phase 皆為獨立 Docker Compose 模組，帶自己的 `deploy.sh`，可逐層部署或一鍵全部部署：

| Phase | 名稱 | 包含服務 |

|-------|------|---------|

| 00 | HWI Advisor | 自動硬體偵測與調校設定檔 |

| 01 | Infrastructure | Portainer, WebSSH |

| 02 | Database | PostgreSQL 17, pgAdmin 4 |

| 03 | AI Interface | Ollama, OpenWebUI, Redis |

| 04 | Automation | n8n queue mode + workers |

| 05 | RAG Stack | Qdrant, Docling, Mosquitto |

| 06 | AI Core Engine | Lemonade inference engine |

| 07 | Validation | Health checks, benchmark scripts |

| 08 | Backup & Recovery | 一鍵備份、還原、VRAM purge |

| 09 | Monitoring & Alerts | tiger-monitor, MQTT alerting |

| 10 | Observability | Grafana, Prometheus, Loki, cAdvisor, DCGM Exporter |

| 11 | Lifecycle | What's Up Docker (WUD) |

---

核心功能

  • **🖥️ Multi-GPU 支援**：AMD ROCm + NVIDIA CUDA + ARM64 同一 workflow
  • **🤖 LLM Inference**：Ollama + OpenWebUI + Lemonade（AMD 高效能推理引擎）
  • **📚 RAG Pipeline**：Qdrant 向量資料庫 + Docling 文件處理 + Mosquitto MQTT
  • **⚡ Workflow Automation**：n8n 佇列模式 + Redis + 分散式 workers
  • **📊 Full Observability**：Grafana + Prometheus + Loki + cAdvisor + DCGM GPU 遙測
  • **🔧 Auto Hardware Tuning**：HWI Advisor 自動偵測 CPU/GPU 並寫入最佳化調校設定檔
  • **💾 One-Click Backup**：所有持久化資料的時間戳備份/還原 + VRAM 清除

AI Agent 原生整合

OpenGenie 為 LLM 和自主 Agent 設計了完整的文件索引和部署協議：

  • **`llms.txt`**：標準化索引檔案，放在專案根目錄，任何 LLM（Claude、ChatGPT、Gemini、Perplexity 等）可直接抓取整個專案的文档、技能和 12 階段模組
  • **6 個 Agent Skills**（自主部署協議）：

- `00-Master orchestrator`：主控制器

- `01-Deployment state machine`：部署狀態機

- `02-Error recovery guide`：錯誤復原指南

- `03-GPU robustness`：GPU 穩定性強化

- `04-Full purge procedure`：完整清除程序

- `05-Installation guide`：安裝指南

**狀態機含磁碟持久化復原**，可支援系統重啟、網路中斷、部署中斷等异常情境恢復。

評論

OpenGenie 是目前看到**最完整的自託管 AI 基礎設施開源框架之一**。核心亮點：

  • Docker Compose 模組化設計，自由度極高
  • 12 階段的狀態機部署協議明確，無需自行摸索流程
  • `llms.txt` 讓任何 AI Agent 可以直接讀取、理解並執行部署
  • 同時支援 AMD/NVIDIA/ARM64，適合有多 GPU 需求的實驗室或中小團隊
