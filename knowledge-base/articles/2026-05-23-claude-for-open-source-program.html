Claude for Open Source：OSS 維護者可申請 6 個月 Claude Max

一句話摘要

Anthropic 推出 Claude for Open Source Program，符合資格的開源維護者與貢獻者可申請 6 個月免費 Claude Max 20x，用來感謝與支持 OSS 生態系的核心貢獻者。

重點整理

  • 官方頁面標題：Claude for Open Source。
  • 方案內容：符合資格者可獲得 6 個月免費 Claude Max 20x。
  • Anthropic 的定位是感謝 open-source maintainers and contributors 對生態系的貢獻。
  • 申請採 rolling basis 持續審核。
  • 官方提到最多接受 10,000 位 contributors。
  • 若申請通過，會收到啟用 Claude Max 的連結，可在帳號上啟用訂閱期間。
  • 官方列出的主要申請對象：

- public repo 的 primary maintainer 或 core team member。

- GitHub repo 需有 5,000+ stars，或 NPM 每月下載量 1M+。

- 最近 3 個月內有 commits、releases 或 PR reviews。

  • 官方也補充：如果不完全符合條件，但維護的是生態系默默依賴的重要專案，也可以申請並說明原因。
  • 申請表欄位包含：first name、last name、email、GitHub handle、GitHub repository URL、other info。

  • 可追蹤身邊或課程社群中是否有人維護符合資格的 OSS 專案，提醒他們申請。
  • 若未來 Luarn-Pin 或合作社群有開源專案達到門檻，可準備：
