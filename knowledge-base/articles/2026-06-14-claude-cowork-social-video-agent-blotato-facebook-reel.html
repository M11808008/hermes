Claude Cowork × Blotato｜用桌面 Agent 生成並自動發布社群影片的工作流

來源為 Sabrina Ramonov 的 Facebook Reel。貼文主張：留言「COWORK」即可取得免費 Claude Cowork masterclass；範例展示 Claude Cowork 能讓影片自動生成、再發布到社群，而且不需要額外影像或影片生成 credits，因為流程在本機透過 Cowork 與免費 Python 函式庫完成；最後可接 Blotato 自動發到多個社群平台。本文把 Reel 視為社群發現來源，並以 Anthropic Claude Cowork 官方頁、Blotato 官方頁與 Sabrina 的 Cowork 教學頁交叉驗證。

一句話摘要

這支 Reel 值得收進 AI 學習雷達，因為它不是單純「AI 生成影片」炫技，而是展示一個可複製的 creator-agent 架構：Claude Cowork 負責讀取本機素材、編排腳本與呼叫 Python 工具生成影片；Blotato 負責把完成內容排程並分發到 TikTok、Instagram、YouTube、Threads、Facebook 等平台。

可行工作流拆解

  • 素材準備：把品牌圖片、照片、短片、b-roll、logo、音檔與文字素材放在 Cowork 可存取的本機資料夾。
  • 腳本與分鏡：讓 Cowork 依主題產生短影音腳本、hook、畫面段落、字幕與 CTA。
  • 本機生成：透過 Python libraries 組合圖片、片段、字幕、轉場與輸出比例，產生可上傳影片。
  • 品質檢查：人工確認字幕、品牌語氣、版權、錯字、畫面節奏與平台規範。
  • 分發排程：把影片與文案交給 Blotato 或其 API/MCP/自動化整合，安排跨平台發布。
  • 迭代回饋：用平台成效資料調整 hook、模板、長度、封面與 CTA。

建議導入順序

| 階段 | 做法 | 目的 |

|---|---|---|

| 1. 純本機 prototype | 只用 3–5 張圖片、字幕與背景音，讓 Python 產出一支短片 | 驗證 Cowork 能否穩定讀寫素材與輸出影片 |

| 2. 固定模板 | 建立 1–2 個品牌模板：標題、字幕、logo、結尾 CTA | 避免每次從零做設計 |

| 3. 人審 gate | 發布前固定輸出待審資料夾，不直接自動上架 | 降低錯字、誤導與品牌風險 |

| 4. 接 Blotato | 只先接一個低風險平台測試排程 | 驗證 API、格式、封面、文案與帳號權限 |

| 5. 擴到多平台 | 依平台調整比例、長度、文案與 hashtag | 避免同一素材硬貼所有平台 |

| 6. 成效迭代 | 回收觀看、留言、分享、點擊資料 | 讓 Agent 依數據改 hook 與模板 |

風險與限制

  • 「unlimited videos」應解讀為本機自動化理論上可大量生成，不代表沒有時間、硬體、素材、平台 API、發布配額與品質成本。
  • 免費 Python libraries 可以做字幕、剪接、排版、合成與轉檔，但若需要高品質 AI 影像、AI 影片或 voice cloning，仍可能需要額外模型或雲端服務。
  • 自動發布工具會接觸社群帳號權限；應避免讓 Agent 直接發布未審核內容。
  • Reel 沒有展示完整程式碼、素材來源、模板細節與 Blotato 設定，因此本文只能確認架構方向，不能保證照貼文操作即可一次成功。
  • 若內容使用他人照片、影片、音樂或新聞素材，必須先處理授權與平台政策。

欒帥版落地建議

優先把它做成「AI 學習雷達短影音再利用」的小型流水線：從知識庫 Markdown 選一篇高價值文章，輸出 45 秒腳本、五張視覺卡、字幕檔與影片；人工確認後再交給 Blotato 或其他排程工具。這樣能把既有知識庫內容轉成社群素材，同時保留品質與品牌審核，不必一開始追求全自動大量發布。
