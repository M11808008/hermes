Agent 工具地圖 — 從個人工作代理到開發者工作台

一句話摘要

AI 工具正在從聊天介面，進化成能理解目標、拆解任務、使用工具並持續執行工作的 Agent 系統。

重點整理

  • Personal / Workspace Agent：Gemini Spark、Claude Cowork 代表的是個人與工作區代理，偏向郵件、文件、行事曆、資料整理與報告產出。
  • Coding Agent / Agent Command Center：Codex 與 Antigravity 2.0 的重點是軟體開發流程，能協調 editor、terminal、browser、worktrees 與多代理平行任務。
  • Agent Framework：OpenClaw 與 Hermes Agent 不是單純工具，而是開發者用來打造 persistent agent 的框架。
  • OpenClaw 強調長任務、工具使用、記憶與工作流自動化。
  • Hermes Agent 強調 learning loop、skills、memory 與跨 session 學習能力。
  • 選工具時可先判斷：要直接使用 Agent 提升效率，還是要打造自己的 Agent 系統。

對 Luarn-Pin 的意義

這篇可作為「AI 從聊天工具走向工作系統」的課程或簡報開場素材。它也剛好把 Hermes Agent 放在 Agent Framework 的脈絡中，適合作為個人知識庫中 AI Agent 主題的第一筆核心筆記。
