pyJianYingDraft｜用 Python 生成剪映草稿，串接 WorkBuddy 的半自動剪輯工作流

Threads 貼文指出：「Python 有一個包，可以建立剪映草稿，接上 WorkBuddy 就可以自動剪輯。」留言補充對應 GitHub 專案為 `GuanYixuan/pyJianYingDraft`。本文將社群發現整理成可實作的技術筆記，並用 GitHub README / API metadata 校正限制。

一句話摘要

pyJianYingDraft 不是直接替你完成所有創意剪輯，而是讓 Python 生成剪映草稿與時間線；若前面接上 WorkBuddy、Hermes、Codex 或其他 agent，就能把「請 AI 規劃怎麼剪」轉成草稿檔，再交給剪映檢查、微調與匯出。

欒帥可用的半自動剪輯流程

  • 素材整理：將原始影片、圖片、旁白、音樂、字幕檔放入固定資料夾。
  • AI 規劃：請 Hermes / WorkBuddy / Codex 讀素材清單與腳本，輸出剪輯企劃：片段順序、起訖時間、字幕、B-roll、音樂點。
  • Python 生成草稿：由 pyJianYingDraft 把企劃轉成剪映草稿時間線。
  • 剪映人工審稿：在剪映中打開草稿，檢查節奏、字幕、轉場、音量與素材授權。
  • 匯出與發布：目前若要穩定自動導出，優先考慮 Windows + 受支援剪映版本；若在 macOS，先以草稿生成 + 人工匯出為主。

適合做成的內容自動化

| 場景 | 做法 |

|---|---|

| 課程短影音 | AI 依講義產生 60–90 秒片段腳本，再生成字幕與鏡頭時間線 |

| 工具教學影片 | 固定片頭 / 片尾 / 字幕樣式作為模板，替換螢幕錄影與關鍵字 |

| 社群精華剪輯 | 將長影片切成多個主題短片草稿，人工挑選後匯出 |

| AI 學習雷達週報影片 | 從知識庫週報選 3–5 則重點，自動生成口播字幕與素材位置 |

建議先做的最小實驗

  • 安裝：`pip install pyJianYingDraft`。
  • 找到剪映草稿資料夾，例如 `.../JianyingPro Drafts`。
  • 跑官方 `demo.py`，確認剪映中能看到新草稿與時間線。
  • 先只做「素材 + 字幕 + 轉場」三件事，不要一開始追求全自動剪輯。
  • 成功後再讓 agent 產生 JSON 剪輯計畫，Python 只負責把計畫轉為草稿。
