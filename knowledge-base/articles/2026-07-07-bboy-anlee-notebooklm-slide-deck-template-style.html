李安｜NotebookLM 固定背景風格簡報：用模板來源與設定降低隨機設計

一句話摘要

NotebookLM 生成簡報時若風格太隨機，可以把模板 / 品牌規範當成來源，並在 Slide Deck 設定中明確指定風格，讓 AI 產出的簡報更像同一套設計系統。

李安這則 Threads 的主文很短，核心操作是：

  • NotebookLM 可製作固定背景風格簡報。
  • 如果不希望每次生成都是隨機風格，可以在資料來源上傳自己的模板。
  • 在簡報設定介面指定模板，生成更一致的簡報。

官方資料交叉驗證

Google NotebookLM Help 的「Generate a Slide Deck in NotebookLM」說明：

  • Slide Deck 可把 sources 轉成 polished presentation。
  • 使用者需在 notebook 中上傳 sources。
  • 在 Studio panel 選擇 Slide Deck，或用 pencil icon 在生成前自訂。
  • customization panel 可設定 format、output language、length，也可以輸入 prompt 描述想要的簡報。
  • prompt 可指定 audience、style、focus；官方範例為「Create a deck for beginners using a bold and playful style with a focus on step-by-step instructions.」
  • Slide Deck 可在 NotebookLM 內播放，也可下載為 PDF。
  • AI 生成簡報可能有視覺或事實錯誤，需要檢查。

Google NotebookLM sources help 也驗證 NotebookLM 支援 Google Slides 作為來源，且每個 Google Slides 來源最多 100 slides；同時也支援 PDF、PowerPoint、Google Docs、web URLs、YouTube、圖片、音訊等來源。

Google Blog 的 Slide Decks 使用建議進一步提到：NotebookLM 可把 dense research、rough notes、brainstorm 轉成 custom slide decks，也可以用 project sources 加入 brand styling 與 visuals。這與貼文所說「把模板上傳為資料來源，再指定模板」的做法方向一致。

這個技巧真正解決的問題

NotebookLM 的 Slide Deck 很適合快速把資料變成簡報，但如果每次都只讓模型自由發揮，常見問題是：

  • 背景與配色每次都不同，無法形成品牌一致性。
  • 版型、字級、留白、標題層級不穩定。
  • 教學簡報或企業簡報需要固定識別，但 AI 可能改成隨機視覺。
  • 同一系列課程 / 社群教學影片需要風格一致，否則看起來像不同來源拼接。

把模板、品牌規範或上一份成功簡報當作 NotebookLM source，再在 Slide Deck prompt 中指定「依照此模板的背景、色彩、版面密度與標題風格」，等於給 NotebookLM 一個風格參考錨點，降低每次重抽設計的隨機性。

建議操作流程

可把這則貼文整理成一個可複製流程：

  • 準備資料內容來源：文件、文章、研究資料、課程大綱、影片逐字稿。
  • 準備模板來源：Google Slides / PowerPoint / PDF，內含品牌色、背景、標題頁、章節頁、內容頁樣式。
  • 上傳到同一個 NotebookLM notebook。
  • 在 sources panel 勾選「內容來源」與「模板來源」。
  • 到 Studio 選 Slide Deck。
  • 在 customization prompt 中明確寫：

- 請根據內容來源生成簡報。

- 視覺風格請參考模板來源。

- 保持固定背景、色彩、字級階層、頁首 / 頁尾與留白邏輯。

- 不要發明模板中沒有的品牌元素。

- 若模板與內容衝突，優先保持可讀性。

  • 生成後下載 PDF 或在 NotebookLM 檢視。
  • 人工檢查：事實、引用、中文字、圖表、對齊、頁面密度與品牌一致性。

可用提示詞範例

```text

請根據已選取的內容來源製作一份簡報。視覺風格請參考「簡報模板」來源：保持相同背景風格、主色與輔色、標題層級、留白比例、頁碼 / 頁尾邏輯與章節分隔方式。請不要每頁生成完全不同的風格。若需要新增圖像或視覺元素，請優先使用與模板一致的扁平化 / 簡潔風格。輸出對象是初學者，內容要清楚、分段、有教學節奏。

```

使用限制與注意事項

  • 官方文件確認可以用 prompt 指定 style，但 NotebookLM 是否能完全照著模板輸出，仍取決於模型理解與當下功能狀態。
  • Slide Deck 目前可能產生視覺或事實錯誤，不能直接當成最終交付版。
  • 若模板含商標、受保護圖片、客戶資料或內部簡報，需確認有權上傳到 NotebookLM。
  • 官方說明提到 revision 階段目前 sources 不會被納入修訂考量；若要大幅改風格，可能要重新生成，而不是只靠逐頁 revision。
  • NotebookLM 輸出多為 PDF / 內部播放；若需要可完全編輯的 PPTX，可能還要搭配 Google Slides、PowerPoint 或其他工具後製。
