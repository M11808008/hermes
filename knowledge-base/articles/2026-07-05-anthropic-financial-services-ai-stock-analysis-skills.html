Anthropic Financial Services｜AI 金融技能庫：把財報季、股票研究與價值投資工作流指令化

一句話摘要

AI 正在把金融研究從「人工讀資料、手動整理、慢慢建模型」推向「資料收集、財報判讀、估值模型、投資論點與催化事件追蹤」的技能化工作流；Anthropic `financial-services` repo 是目前很值得研究的官方範例。

為什麼這件事重要

近期 AI 圈開始快速往金融市場發展，特別是股票研究與基本面分析。這不只是把財報摘要得更快，而是把價值投資者常做的研究流程拆成可重複執行的 skill：

  • 財報前先建立預期與情境。
  • 財報後快速辨識 beat / miss 與 thesis impact。
  • 對新標的建立首次覆蓋報告。
  • 用新資料更新財務模型。
  • 維護投資論點與反證條件。
  • 追蹤催化事件與每日盤前想法。

換句話說，AI 不只是讀資料，而是把研究流程「機構化」與「指令化」。

1. 財報季的「降維打擊」武器

在美股或台股財報密集週，資訊量往往大到讓人窒息。最關鍵的兩個指令是：

`/earnings-preview`：財報前瞻

用途：在公司公布財報前，做情境分析與關鍵數據預測。

這個 skill 會幫投資者事先建立：

  • consensus estimates
  • key metrics to watch
  • bull / base / bear scenarios
  • catalyst checklist
  • trading setup
  • option implied move 與歷史財報反應

價值在於：當財報數字真正公布時，投資者能立刻判斷是「超預期」還是「低於預期」，而不是等市場已反應完才開始讀資料。

`/earnings`：財報後更新

用途：財報公布後產出季度更新報告，重點在 beat / miss、財務數字、電話會議重點、guidance 與投資論點變化。

官方 command 描述要求：

  • 先確認資料是否最新。
  • 搜尋 earnings release、10-Q、transcript、investor presentation、consensus estimates。
  • 做 revenue / EPS / segment beat-miss analysis。
  • 更新估值與 forward estimates。
  • 產出 8–12 頁 earnings update report。

這對財報季很有價值，因為它把「讀幾十頁 PDF、對照電話會議、更新模型」變成一條可執行工作流。

2. 機構級的深度研究與論點維護

建立投資部位不能只看線圖，核心仍是基本面研究與持股邏輯。

`/initiate`：首次覆蓋報告

用途：對一檔新股票產出機構等級的首次覆蓋報告。

適合用在：

  • 第一次研究一家公司。
  • 建立完整基本面檔案。
  • 梳理 business model、segment、競爭格局、估值、風險與催化因素。

`/model-update`：更新財務模型

用途：隨著新財報、新 guidance 或新營運數據注入，自動更新模型中的財務參數。

它的關鍵不是「幫你算一次」，而是讓模型與研究判斷能隨資料更新持續同步。

`/thesis`：維護投資論點

用途：定期檢視持股核心邏輯是否仍成立。

價值投資最怕的不是短期股價波動，而是抱著過期或錯誤的期待。`/thesis` 可以協助建立：

  • 核心投資論點
  • 反證條件
  • 風險清單
  • 需要追蹤的 KPI
  • 新資料對 thesis 的正負面影響

3. 每日交易的靈感與催化劑追蹤

這組指令適合盤前準備、選題與日常研究節奏。

`/morning-note`：晨會筆記

用途：快速整理昨夜市場動態，產出今日交易想法與關注重點。

適合金融 KOL、投研助理或交易團隊每天早上產出：

  • 市場摘要
  • 個股異動
  • 產業新聞
  • 交易想法
  • 今日待觀察事件

`/sector`：產業研究

用途：從宏觀產業概況、競爭格局、主題趨勢切入，建立研究框架。

適合用來產出：

  • AI infrastructure 產業研究
  • 半導體供應鏈 overview
  • SaaS / cybersecurity / healthcare sector comparison
  • 主題型投資報告

`/screen`：選股篩選

用途：用條件篩選與研究框架找出新的投資想法。

它可作為「想法來源」：從財務條件、估值、成長性、產業位置或催化事件等條件篩出候選標的。

`/catalysts`：催化事件追蹤

用途：追蹤覆蓋股票即將發生的重要事件。

例如：

  • 財報日
  • 法說會
  • 新產品發表
  • 政策公布
  • 併購消息
  • 臨床試驗解盲
  • 分析師日
  • 監管審查節點

這對「提前卡位行情」很重要，因為市場常常不是只反應財報結果，而是反應即將到來的預期變化。

對價值投資者的實際意義

1. 更快建立選股邏輯

過去研究一家公司，需要人工蒐集財報、法說逐字稿、同業資料、估值模型與產業脈絡。現在可以用 skill 讓 AI 先搭出一版研究骨架，再由人修正假設與判斷。

2. 讓研究流程變得可複製

每次研究新股票都可以遵循類似流程：

  • `/screen` 找候選。
  • `/sector` 建立產業脈絡。
  • `/initiate` 做首次覆蓋。
  • `/comps` 與 `/dcf` 做估值。
  • `/thesis` 建立投資論點與反證條件。
  • `/catalysts` 建立事件追蹤。
  • 財報前 `/earnings-preview`，財報後 `/earnings`。

3. 把 KOL 內容從新聞轉述升級成研究產品

對金融 KOL 來說，這套技能可以把內容從「今天發生什麼事」升級成：

  • 這家公司目前市場預期是什麼？
  • 財報結果超預期還是低於預期？
  • 產業趨勢是否支撐原本 thesis？
  • 估值是否反映成長與風險？
  • 接下來有哪些催化事件？

這就是從資訊搬運走向研究型內容。

4. 但最後判斷仍然要由人負責

AI 可以加速資料整理、模型更新與初稿產出，但它不能替代投資責任。尤其金融場景必須特別注意：

  • 資料來源與時間戳。
  • 財務模型假設。
  • consensus estimates 的來源與日期。
  • 估值敏感度。
  • 合規揭露與利益衝突。
  • 不應直接構成個別投資建議。

觀察結論

`anthropics/financial-services` 代表 AI 金融工作流的一個重要方向：不是讓 AI 隨口分析股票，而是把專業投研拆成可審查、可重複、可維護的 skills 與 agents。

對價值投資者而言，這套工具真正提升的是「研究速度」與「流程一致性」：它可以在財報季快速建構前瞻、更新模型、維護論點、追蹤催化事件，讓投資者把更多時間花在判斷假設是否合理，而不是被 PDF、Excel 與逐字稿淹沒。
