GitHub Maigret：用一個用戶名搜尋全網社群帳號

  • 支援的網站類型包含但不限於：

- 社群媒體：Twitter/X、Instagram、Facebook、LinkedIn、Threads、TikTok

- 商業與生产力：Slack、Discord、Telegram、Zoom

- 其他：Pornhub、Steam、各國本地平台等

  • 用途場景：

- OSINT（開源情報）調查：確認某人在網路上的整體存在感

- 帳號安全審計：檢查自己或員工是否有遺漏的舊帳號暴露在外

- 數位足跡分析：了解某個用戶名在網路上的曝光範圍

- 防偽冒：確認特定用戶名是否被搶註冊

  • 屬於被動資訊收集工具，不涉及帳號破解或未授權存取，屬於合法的 OSINT 範疇。
