一句話摘要

這則 Threads 的價值在於提醒：想補 AI 基礎，不一定要先買民間課，Anthropic、OpenAI、Google 都已經把入門到工作流層級的官方學習資源放上線。

原始貼文重點

Aden Chen（@fucell.us）列出三個官方 AI 學習資源：

  • Anthropic 官方免費課（含證書）：https://anthropic.skilljar.com/
  • OpenAI 官方學習資源（OpenAI Academy）：https://academy.openai.com/
  • Google AI Essentials（官方＋證書）：https://www.skills.google/paths/2336

官方來源交叉驗證

1. Anthropic Academy / Skilljar

官方 Skilljar 頁面標題為「Anthropic Courses」，公開列出多門課程，包括：

  • Claude 101
  • Claude Code 101
  • Claude Platform 101
  • Introduction to Claude Cowork
  • Claude Code in Action
  • AI Fluency: Framework & Foundations
  • Building with the Claude API
  • Introduction to Model Context Protocol
  • Model Context Protocol: Advanced Topics
  • Introduction to agent skills

Brave Search 亦找到 Anthropic 官方 learn 頁，描述為「New courses available on Anthropic Academy」，並明確提到可深入學 AI Fluency、API development、Model Context Protocol、Claude Code，完成後可取得 certificates。

2. OpenAI Academy

OpenAI Academy 官方首頁描述：「Unlock the new opportunities of the AI era by equipping yourself with the knowledge and skills to harness artificial intelligence effectively.」頁面包含 Courses、Events、Content、Communities 等入口。

Brave Search 找到 OpenAI Help Center 與 OpenAI 官方文章，確認 OpenAI Academy 課程包含：

  • AI Foundations
  • Applied AI Foundations
  • Agents and Workflows

其中 OpenAI 官方文章〈New OpenAI Academy courses for the next era of work〉說明 Applied AI Foundations 會教使用者把有效 prompt 轉成結構化、可重複的 workflow；OpenAI Help Center 則補充學習者會練習提供 context、review outputs、refine results 與 reuse workflow patterns。

需注意：Threads 相關討論常說「免費、上完有完課證明」，但是否為正式認證應以 OpenAI Academy 當下課程頁與帳號登入後顯示為準。

3. Google AI Essentials

Google Skills 官方頁顯示「Google AI Essentials」為 Google Career Certificates 管理的 Path，頁面顯示：

  • 類型：Path
  • 活動數：1 項活動
  • 課程：Google AI Essentials
  • 預估時長：約 4 小時 45 分鐘
  • 課程定位：提升各職能與產業的 AI 生產力，學習用 AI 產生想法與內容、協助研究與組織、加速日常任務如 email 回覆草稿。

Brave Search 同時找到 Google Skills、Career Skills 與 Coursera 的 Google AI Essentials Specialization。Google Skills 搜尋摘要提到完成 Google AI Essentials 後可取得相關學習成果；Coursera 頁則明確說完成 program 後可取得 Google certificate 並分享至職涯網路。

學習路徑建議

初學者順序

  • 先上 Google AI Essentials：建立通用 AI 使用與生產力語言。
  • 再上 OpenAI Academy 的 AI Foundations / Applied AI Foundations：把 prompt 使用推進到可重複 workflow。
  • 最後上 Anthropic Academy：補 Claude、Claude Code、Claude API、MCP、agent skills 等較偏開發與 agent workflow 的內容。

對工作者的意義

這三個官方資源剛好對應三層能力：

| 層級 | 代表資源 | 能力重點 |

|---|---|---|

| 通用 AI 素養 | Google AI Essentials | 基礎概念、工作效率、日常任務 |

| 工作流設計 | OpenAI Academy | prompt 到 workflow、agents、應用場景 |

| Agent / 開發整合 | Anthropic Academy | Claude Code、API、MCP、Skills、AI Fluency |

可行動清單

  • 想快速補 AI 入門：先收藏 Google AI Essentials。
  • 想把 AI 用進工作流程：優先看 OpenAI Academy 的 Applied AI Foundations 與 Agents and Workflows。
  • 想做 AI agent、Claude Code 或 MCP：直接從 Anthropic Academy 的 Claude Code 101、Building with the Claude API、Introduction to Model Context Protocol、Introduction to agent skills 開始。
