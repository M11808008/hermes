Anthropic Academy｜Claude 13 堂免費課程學習路線：Claude Code、Agent Skills、MCP 與 AI Fluency

數位時代整理 Anthropic Academy 推出的 13 堂免費線上影音課程，範圍涵蓋 Claude 基礎、Claude Code、Agent Skills、Claude API、Model Context Protocol、Amazon Bedrock / Google Vertex AI 雲端部署，以及 AI Fluency 教育與非營利組織應用。多數課程在完成影音與測驗後可取得 Anthropic Academy 官方結業證書。

一句話摘要

這組課程不是單純的 Claude 操作教學，而是一條從「AI 素養」到「Agent 工程」的完整學習路線：非工程背景可先學 4D AI Fluency；工程與自動化工作流則應優先補 Claude Code、Agent Skills、MCP、Claude API 與 RAG。

欒帥建議學習路線

路線 1：教學與課程設計優先

| 順序 | 課程 | 為什麼先學 |

|---|---|---|

| 1 | AI Fluency: Framework & Foundations | 建立「AI 不是代工，而是協作」的共同語言，可直接轉進課程設計 |

| 2 | Teaching AI Fluency | 把 4D 框架轉成課堂鷹架與評量方式 |

| 3 | AI Fluency for Educators / Students | 分別準備教師版與學生版教學材料 |

| 4 | Claude 101 | 作為非工程學員的 Claude 實作入口 |

| 5 | Introduction to agent skills | 將課程中的重複工作流做成可重用技能 |

適合用於欒帥規劃 AI 課程、工作坊、教師研習、學生 AI 素養模組。

路線 2：Agent 工程與自動化優先

| 順序 | 課程 | 為什麼先學 |

|---|---|---|

| 1 | Claude Code in Action | 直接對應 coding agent、檔案操作、PR review、Hooks 與上下文管理 |

| 2 | Introduction to agent skills | 學會把 SOP 寫成 `SKILL.md`，避免每次重貼 prompt |

| 3 | Introduction to MCP | 學會讓 Claude 連接外部工具與資料庫 |

| 4 | MCP Advanced Topics | 補齊權限、雙向通訊、transport 與除錯能力 |

| 5 | Building with the Claude API | 進入 API、RAG、eval、agent 架構與正式產品化 |

| 6 | Bedrock / Vertex AI | 需要企業雲端部署時再補 |

適合用於 Hermes、AI 學習雷達、LINE 連結歸檔、知識庫管理介面、GitHub Pages 驗證、自動化 QA。

對 AI 學習雷達的實務價值

  • **Claude Code + Agent Skills**：可把欒帥常做的歸檔、摘要、部署、驗證流程抽象成可重複 skill，而不是每次重新下指令。
  • **MCP**：適合把知識庫、Google Workspace、Notion、GitHub、LINE/Telegram 等工具變成可控的外部能力。
  • **Claude API / RAG**：可作為未來把 AI 學習雷達升級為查詢型知識系統的基礎。
  • **AI Fluency**：可直接轉成課程語言，尤其是「委託、描述、洞察、勤勉」四個學習維度。
  • **Bedrock / Vertex AI**：可作為企業導入、雲端部署與成本治理課程的進階素材。
  • **Computer Use**：在 Bedrock 課程中被列為 UI 自動化能力，與近期 Gemini Computer Use、CapCut Computer Use 類趨勢可放在同一條「Agent 操作介面」主題線。
