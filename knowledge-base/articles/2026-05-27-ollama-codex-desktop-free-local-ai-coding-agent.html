「免費無限」Ollama + Codex Desktop 本地 AI Coding Agent 實測

**來源：** [Facebook 社團「快樂學習AI」](https://www.facebook.com/groups/346855692144727/posts/3394422154054717/) · 公開社團貼文

---

重點摘要

只要透過 Ollama 官方網站搭配一個指令，就能把 Codex Desktop 改成你自己的「本地 AI 模型」，不只能避免 API 費用，還能直接使用本機 GPU 跑 AI Coding Agent。

---

🚀 一個指令啟動

```bash

ollama launch codex-app

```

立刻把 Codex Desktop 串接到 Ollama！

---

支援模型（推薦）

### Ollama Cloud（進階 AI Agent 任務）

  • `kimi-k2.6`（支援視覺）
  • `glm-5.1`

### 本機模型（免費）

  • `qwen3.6`
  • `gemma4:31b`
  • `nemotron-3-super`

---

直接指定模型啟動

```bash

ollama launch codex-app --model qwen3.6

```

或使用雲端模型：

```bash

ollama launch codex-app --model kimi-k2.6:cloud

```

### 切換回 OpenAI Codex

```bash

ollama launch codex-app --restore

```

---

實測功能

測試任務：「幫我做一個網頁轉 Markdown 的工具」

**結果：真的能跑，且：**

  • ✅ AI 會自動生成程式
  • ✅ 可即時預覽
  • ✅ 能修改需求重新生成

---

實測心得

本機模型速度會比雲端慢一點，但如果你的電腦符合以下條件，體驗已經非常接近雲端 AI：

| 硬體需求 | 建議規格 |

|----------|----------|

| GPU | NVIDIA 獨立顯示卡 |

| RAM | 足夠執行大模型 |

| CPU | 中等以上規格 |

### 適合情境

  • **Vibe Coding**：輕鬆用自然語言生成程式碼
  • **AI Agent 開發**：用本地模型做複雜任務
  • **本地端隱私需求**：程式碼不上雲端
  • **離線 AI 開發**：無網路也能Coding

---

評論

> 以前：「AI 寫程式 = 一直燒 API 費」

> 現在：「本地模型免費無限用，直接變成 Codex Agent」

真的開始有「個人 AI 開發工作站」的感覺了。Ollama 的模型生態持續擴充，加上 Codex 的 Agent 能力，本地 AI Coding 的門檻已大幅降低。
