作者：Ian Peng（@ianpeng1120）

主題：Hermes Agent 推薦串哪個 AI Model 比較省錢又好用？

原貼重點：

他目前用：

deepseek-v4-flash

想問 Hermes Agent 開發者：

有沒有更省錢、效果又好的模型方案？

他也提到：

  • 有訂閱 Gemini Pro
  • 有訂閱 Claude Pro
  • 但 Claude Pro 不能直接當 Hermes Agent API 用，通常還是要 Claude API 另計費
  • Gemini API 也還不熟

底下留言整理重點：

  • 很多人推薦 Codex / OpenAI OAuth

留言普遍認為：

  • 目前最省可能還是 Codex 訂閱制
  • 若 Hermes 可用 OpenAI OAuth 串 GPT-5.5，是目前很強的方案之一
  • 有人說 Codex 好用，甚至可以生圖

這其實跟您目前的配置一致：

您現在 Hermes 是 OpenAI Codex OAuth，模型 gpt-5.5。

  • DeepSeek v4 flash / pro

留言意見：

  • deepseek-v4-flash 已經算省
  • 但有人建議不要用 flash，要用 deepseek-v4-pro 效果比較好
  • 也有人說 deepseek v4 flash 推理開高其實不錯
  • 目前到五月底 deepseek 有折扣，價格很香
  • OpenCode Go / Opencode Go

有人推薦：

  • opencode go
  • deepseek-v4-pro
  • 價格約 5～10 美金/月
  • 適合「便宜大碗」路線
  • MiniMax

留言提到：

  • MiniMax token plan 很吃飽
  • MiniMax M2.7 便宜、量大
  • 但智力略低於 DeepSeek v4 flash/pro
  • 可當便宜大量任務模型
  • Gemini Pro 訂閱 ≠ Gemini API

有人特別提醒：

Gemini Pro 訂閱跟 Gemini API 是不同東西。

也就是說：

您有 Gemini Pro，不代表 Hermes Agent 可以直接免費用 Gemini API。

如果要接 Hermes，多半還是要用 Gemini API key，按 token 計費。

  • Claude Pro 訂閱也類似

Claude Pro 訂閱通常不能直接當 Hermes API 使用。

要給 Hermes Agent 用 Claude，通常要：

  • Anthropic API key
  • 另外按 API 用量計費
  • 免費或低價選項

留言也提到：

  • Google AI Studio 免費額度可試
  • OpenRouter 上標 free 的模型可以玩
  • NVIDIA NIM 可以試模型
  • 中介平台 API 可能便宜，但穩定性與安全性需自行評估

我的判斷，依您的情境：

您現在最佳主力其實就是：

OpenAI Codex OAuth + GPT-5.5

理由：

  • 您已經登入成功
  • 不需要另外 API token 燒錢
  • Hermes 狀態顯示目前就是這個 provider
  • 對工具調用、程式修復、長任務執行，比多數便宜模型穩

如果要省錢分層，我會建議：

主力高品質：

OpenAI Codex OAuth / GPT-5.5

大量便宜任務：

MiniMax M2.7 或 DeepSeek v4 flash

較聰明但仍相對省：

DeepSeek v4 pro

測試免費模型：

Google AI Studio / OpenRouter free / NVIDIA NIM

不建議您現在為了省一點錢，把主力 Hermes 改掉。因為您這台已經接 LINE Gateway、檔案處理、PDF 修正、Codex OAuth，穩定性比單純便宜更重要。

最實際策略是：

「主模型維持 GPT-5.5 Codex OAuth，輔助/大量任務再接便宜模型。」
